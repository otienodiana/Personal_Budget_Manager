# Budget Tracker App

A web application for tracking personal expenses and managing budgets efficiently. Users can set budget limits, add expenses, and receive alerts when they exceed their limits.

## Features

- **Add and Categorize Expenses**: Easily input and categorize your expenses to keep track of spending.
- **Set Budget Limits**: Define budget limits for different expense categories to manage finances effectively.
- **Receive Alerts**: Get notifications when your expenses exceed budget limits.
- **Expense Summary**: View a detailed summary of all your expenses for better financial planning.

## Technologies Used

- **Frontend**: React
- **Routing**: React Router
- **State Management**: React's built-in `useState` and `useEffect` hooks
- **Local Storage**: For persistent storage of alerts
- **Styling**: CSS for styling components
